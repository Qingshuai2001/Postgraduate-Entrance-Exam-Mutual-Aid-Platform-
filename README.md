## 📖 项目简介

🎓 随着教育行业的蓬勃发展和信息技术的快速进步，通过互联网平台获取高质量学习资料对考研学子变得至关重要。然而，当前网络教育平台普遍存在以下问题：

- ❌ 信息质量良莠不齐
- 📢 垃圾广告泛滥成灾
- 🎨 界面设计单调乏味
- 💰 优质资料收费高昂

💡 为解决上述痛点，我们开发了**考研互助平台**，旨在为考研学子提供一个纯净、高效、实用的学习与交流环境，显著提升备考效率。

## ✨ 功能特性

| 功能模块 | 图标 | 描述 |
|---------|------|------|
| 📰 **最新资讯** | 📰 | 实时更新考研政策、院校动态等权威信息 |
| 📚 **资料共享** | 📚 | 免费获取高质量学习资料，支持上传下载 |
| 🛒 **二手交易** | 🛒 | 考研书籍、资料循环利用，节约备考成本 |
| 💬 **研友论坛** | 💬 | 与全国研友交流心得，互助解惑 |
| 🏆 **经验分享** | 🏆 | 上岸学长学姐真实经验，助力高效备考 |

## 🛠️ 技术栈

### 前端技术
- **框架**:  Vue.js - 数据绑定与交互
- **UI库**:  ElementUI - 界面组件库
- **开发工具**:  Visual Studio Code

### 后端技术
- **语言**:  Java - 后端开发
- **数据库**:  MySQL - 数据存储
- **开发工具**:  IntelliJ IDEA

### 开发方法
- 🔧 **面向对象软件工程方法** - 系统设计与开发

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/Qingshuai2001/Postgraduate-Entrance-Exam-Mutual-Aid-Platform-.git

# 进入项目目录
cd postgraduate-platform

# 安装依赖
npm install

# 启动开发服务器
npm run serve

# 构建生产版本
npm run build
```
